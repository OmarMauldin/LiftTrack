# LiftTrack
 
A full-stack fitness tracker web app built as a portfolio project. LiftTrack lets users create an account, build custom workout templates, log sets in real time, and track their progress over time.
 
Currently in active development.
 
## Features
 
- User authentication — sign up, sign in, sign out
- Create and manage custom workout templates
- Active workout session — log sets, mark complete or failed, track rest
- Workout history saved per user
- Dashboard with stats — total volume, workouts this week, estimated 1RM
- Dark mode UI built to a custom Figma design
## Tech Stack
 
- React 19
- Vite 7
- Tailwind CSS 4
- shadcn/ui
- React Router 7
- Supabase — auth and PostgreSQL database
- Inter Variable font
